About

Blogs
Displays all the blogs. Pagination is applied, and links are provided to navigate to the pages of blogs. Each blog includes display picture of the author , link to all the posts of the author of post, title of the post and short description if provided by the author.
Login and SignUp
Registered users can login and the other users can SignUp using the links given on the navigation bar.
Profile
For every registered user a profile page is there. Users can update their profile from here.The profile page is visible only to the user of that profile.
Post page
Every post has a page for it. The author of the post can update and delete their post from this page. For updating a post the user will be directed to a page with a form to update their post. The registered users can comment on the post.All users can view the content of the post and comments on that post on this page.
Search
Users can search for any post on the website by either the title of that post or the author of that post. Search field is available on the navigation bar.
